Chapitre 1 : Algorithmes Gloutons

Principe Général

Un algorithme glouton construit une solution étape par étape. À chaque étape, il fait le choix qui semble le meilleur localement (le plus "prometteur" sur le moment) dans l'espoir d'atteindre une solution optimale globale.

Attention : L'approche gloutonne ne garantit pas toujours de trouver la meilleure solution globale pour tous les problèmes. Mais pour certains, comme le rendu de monnaie avec des systèmes de pièces "canoniques" (comme l'Euro ou le Dollar US), elle fonctionne et est efficace.

Exemple : Le Problème du Rendu de Monnaie

Objectif : Rendre une somme d'argent donnée en utilisant le moins de pièces/billets possible, à partir d'un ensemble de pièces/billets disponibles.

Stratégie Gloutonne :

  1. Trier les dénominations de pièces/billets disponibles par ordre décroissant (si ce n'est pas déjà fait).
  2. Tant que le montant à rendre est supérieur à zéro :
    • Prendre la plus grande dénomination de pièce/billet qui est inférieure ou égale au montant restant à rendre.
    • Ajouter cette pièce/billet à la solution.
    • Soustraire la valeur de cette pièce/billet du montant restant à rendre.

Important : Cette stratégie gloutonne fonctionne pour les systèmes de monnaie standards (ex: {1, 2, 5, 10, 20, 50} centimes, {1, 2} euros). Elle pourrait ne pas fonctionner pour des systèmes de pièces "exotiques" (ex: pièces {1, 3, 4} pour rendre 6. Glouton : 4+1+1=3 pièces. Optimal : 3+3=2 pièces). Pour le DS, concentre-toi sur le cas où ça marche.

Programme Python : Rendu de Monnaie Glouton 

def renduMonnaie(somme, monnaie) :
i = 0
L = []
while somme > 0 :
    if somme >= monnaie[i] :
    somme = somme - monnaie[i]
    L.append(monnaie[i])
else :
    i += 1
return L

Chapitre 2 : Recherche Dichotomique

Principe Général

La recherche dichotomique (ou Binary Search) est un algorithme de recherche très efficace pour trouver un élément dans une liste triée.

Son principe est de "diviser pour régner" :

  1. Comparer l'élément recherché avec l'élément au milieu de la liste.
  2. Si c'est l'élément recherché, on l'a trouvé.
  3. Si l'élément recherché est plus petit, on élimine la moitié supérieure de la liste et on recommence la recherche sur la moitié inférieure.
  4. Si l'élément recherché est plus grand, on élimine la moitié inférieure de la liste et on recommence la recherche sur la moitié supérieure.
  5. On répète ce processus jusqu'à trouver l'élément ou jusqu'à ce que la portion de liste à examiner soit vide (l'élément n'est pas présent).

Prérequis fondamental : La liste doit être triée !

Efficacité : Très rapide, complexité en O(log n), où n est la taille de la liste.

Visualisation de la Recherche Dichotomique

Programme Python : Recherche Dichotomique 

def recherche_dichotomique(liste_triee, element_recherche):
    """
    Recherche un élément dans une liste triée en utilisant la recherche dichotomique.
    
    Args:
        liste_triee (list): La liste dans laquelle chercher. Doit être triée !
        element_recherche: L'élément à trouver dans la liste.
    
    Returns:
        int: L'index de l'élément s'il est trouvé, sinon -1.
    """
    debut = 0                      # Index du premier élément
    fin = len(liste_triee) - 1     # Index du dernier élément
    
    while debut <= fin:
        # Calculer l'index du milieu (utiliser // pour une division entière)
        milieu = (debut + fin) // 2
        
        # Vérifier si l'élément au milieu est celui qu'on cherche
        if liste_triee[milieu] == element_recherche:
            return milieu  # Trouvé ! Retourne l'index.
        
        # Si l'élément recherché est plus petit, chercher dans la moitié gauche
        elif element_recherche < liste_triee[milieu]:
            fin = milieu - 1  # Le nouvel index de fin est juste avant le milieu
        
        # Sinon (l'élément recherché est plus grand), chercher dans la moitié droite
        else:
            debut = milieu + 1  # Le nouvel index de début est juste après le milieu
    
    # Si la boucle se termine sans trouver l'élément (debut > fin)
    return -1  # Non trouvé

# Exemple d'utilisation
ma_liste = [2, 5, 8, 12, 16, 23, 38, 56, 72, 91]  # Liste déjà triée
element1 = 23
element2 = 40

index1 = recherche_dichotomique(ma_liste, element1)
index2 = recherche_dichotomique(ma_liste, element2)

if index1 != -1:
    print(f"L'élément {element1} se trouve à l'index {index1}")
else:
    print(f"L'élément {element1} n'est pas dans la liste.")

if index2 != -1:
    print(f"L'élément {element2} se trouve à l'index {index2}")
else:
    print(f"L'élément {element2} n'est pas dans la liste.")

Chapitre 3 : k-Plus Proches Voisins (kNN) - Focus sur la Distance

Principe Général de kNN

L'algorithme des k-plus proches voisins (kNN) est un algorithme d'apprentissage supervisé utilisé pour la classification ou la régression.

Pour classer un nouveau point de données :

  1. Calculer la distance entre le nouveau point et tous les autres points du jeu de données d'entraînement.
  2. Identifier les k points d'entraînement les plus proches (les "k voisins").
  3. Pour la classification : Attribuer au nouveau point la classe la plus fréquente parmi ses k voisins (vote majoritaire).
  4. Pour la régression : Attribuer au nouveau point la moyenne des valeurs de ses k voisins.

Le calcul de la distance est une étape cruciale. La distance la plus couramment utilisée est la distance Euclidienne.

Calcul de la Distance Euclidienne

La distance Euclidienne entre deux points P = (p₁, p₂, ..., pn) et Q = (q₁, q₂, ..., qn) dans un espace à n dimensions est donnée par la formule :

Distance(P, Q) = √[ (q₁ - p₁)² + (q₂ - p₂)² + ... + (qn - pn)² ]

Autrement dit : on calcule la racine carrée de la somme des carrés des différences entre les coordonnées correspondantes des deux points.

Programme Python : Calcul de la Distance Euclidienne 

import math  # Importer le module math pour utiliser la racine carrée (sqrt)

def distance_euclidienne(point1, point2):
    """
    Calcule la distance euclidienne entre deux points dans un espace à n dimensions.
    
    Args:
        point1 (list ou tuple): Les coordonnées du premier point (ex: [x1, y1, z1]).
        point2 (list ou tuple): Les coordonnées du second point (ex: [x2, y2, z2]).
    
    Returns:
        float: La distance euclidienne entre les deux points.
               Retourne None si les points n'ont pas le même nombre de dimensions.
    """
    # Vérifier que les points ont le même nombre de dimensions
    if len(point1) != len(point2):
        print("Erreur : Les points doivent avoir le même nombre de dimensions.")
        return None
    
    somme_carres_differences = 0
    # Parcourir toutes les dimensions (coordonnées)
    for i in range(len(point1)):
        difference = point1[i] - point2[i]
        somme_carres_differences += difference ** 2  # Ajouter le carré de la différence
    
    # Calculer la racine carrée de la somme
    distance = math.sqrt(somme_carres_differences)
    return distance

# Exemple d'utilisation en 2D
point_A = [1, 2]
point_B = [4, 6]
dist_AB = distance_euclidienne(point_A, point_B)
print(f"Distance entre {point_A} et {point_B} (2D) : {dist_AB:.4f}")
# Attendu: sqrt((4-1)^2 + (6-2)^2) = sqrt(3^2 + 4^2) = sqrt(9 + 16) = sqrt(25) = 5.0

# Exemple d'utilisation en 3D
point_C = [1, 0, 5]
point_D = [2, 2, 3]
dist_CD = distance_euclidienne(point_C, point_D)
print(f"Distance entre {point_C} et {point_D} (3D) : {dist_CD:.4f}")
# Attendu: sqrt((2-1)^2 + (2-0)^2 + (3-5)^2) = sqrt(1^2 + 2^2 + (-2)^2) = sqrt(1 + 4 + 4) = sqrt(9) = 3.0

# Exemple avec des dimensions différentes (doit retourner une erreur)
point_E = [1, 1]
point_F = [1, 1, 1]
print(f"Tentative de calcul entre {point_E} et {point_F} :")
dist_EF = distance_euclidienne(point_E, point_F)
if dist_EF is None:
    print("Calcul impossible comme attendu.")
